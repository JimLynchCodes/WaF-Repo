# WaF-Repo
Mono-repo for the With-A-Friend App.

## Node Version
Please use node version 12.14.1
If you have [nvm](https://github.com/nvm-sh/nvm) installed globaly, you can run:
```
nvm i
nvm use
```

# Yarn
please use yarn to install deps:
```
yarn install
```

start local dev server:
```
yarn start
```
